## ⚡ Installation & Setup

Clone the repository:

```bash id="mern1"
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### Install dependencies

```bash id="mern2"
# Backend
cd backend
npm install

# Frontend
cd ../frontend
npm install
```

### Run the project

```bash id="mern3"
# Run backend
cd backend
npm start

# Run frontend
cd ../frontend
npm start
```

---

## 🌍 Environment Variables

Create a `.env` file in the backend folder and add:

```bash id="mern4"
MONGO_URI=your_mongodb_connection_string
PORT=5000
```
